# 改良版ターム分けシステム説明書

## 入力ファイル(CSV)
以下の内容をグループぶん繰り返してください。  
現時点では、グループ番号は0-indexedの整数しか対応していません。  
一行に書く内容:
> グループ番号,人数,第1希望ターム,第2希望ターム,・・・,第12希望ターム

## 出力ファイル(CSV)
各行について,で区切られた要素が3つあるので、左からS1,S2,S3とします。
+ S1 が `0_` で始まるとき その行は無視してください
+ S2 が `3_` で始まるとき その行は無視してください
+ S3は無視してください
+ S1 の `_` の後の数字を A とします。
+ S2 の `_` の後の数字を B とします。

このとき、出力はグループAはタームB(0-indexed)に割り当てられたことを示します。

## 起動
> java (Main.javaのパス)

## 注意
タームの人数上限比率設定、優先順位、二分探索はまだ実装されていません。
